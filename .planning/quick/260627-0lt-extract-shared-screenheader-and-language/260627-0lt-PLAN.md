# Quick Task 260627-0lt — Extract ScreenHeader + LanguageToggle

## Goal

Behavior-preserving refactor. Eliminate the ~6 bespoke header systems and 6 copy-pasted
language-toggle pills by extracting two shared components and routing all 9 screens through
them. **No user-facing copy or callback behavior changes.** Only standardize values to the
existing design tokens.

## Components

### `src/components/LanguageToggle.tsx`
- The မြန်မာ/ENG segmented pill, canonical styling (padding `7px 12px` — not Profile's drifted `7px 13px`).
- Props: `lang: Lang`, `onChange: (l: Lang) => void`, `track?: 'surface' | 'bg'` (default `surface` = `#fff`; `bg` = `var(--color-bg)` for the Profile Settings card, preserving its exact look).
- Named + default export, JSDoc.

### `src/components/ScreenHeader.tsx`
- Props: `variant: 'brand' | 'nav'`, `title?: string`, `align?: 'left' | 'center'` (default `left`), `onBack?: () => void`, `right?: ReactNode`, `divider?: boolean`.
- ONE top padding `24px 20px 16px`. ONE nav-title size 18/600 (`var(--font-sans)` so Burmese→Noto, Latin→Inter, identical to current per-lang fonts). ONE 40px back button (arrow icon, `text-primary`).
- `brand`: blood-drop SVG + "Blood Help" wordmark; `align` left (default) | center; optional `right` slot (toggle / bell).
- `nav`: back (or 40px spacer) + centered title (or empty) + right slot (or 40px spacer to balance).
- `divider?` adds a 1px bottom border — preserves the form screens' fixed-header/scroll separation (per report's "preserve visually via the component").

- LanguageToggle padding → `7px 12px` everywhere (Profile was `7px 13px`).
