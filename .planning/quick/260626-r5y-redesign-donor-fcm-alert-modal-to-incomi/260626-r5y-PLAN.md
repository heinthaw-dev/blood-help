# Quick Task 260626-r5y — Donor FCM alert modal redesign

## Goal

Replace the donor FCM alert (the `fcmDonorAlert` overlay currently inline in
Home.tsx) with the Claude Design **"Incoming Request Alert"**: a centered,
two-state modal whose phone number is gated behind "I'll help".

## Why / context

- Modal currently lives inline in `src/screens/Home.tsx` as a bottom sheet, fed
  by `fcmDonorAlert = { requestId, bloodType, urgency, address }`.
- The new design needs **units, distance, phone** too. The FCM payload doesn't
  carry these, but RLS `active_requests_select` lets any authenticated user read
  active `blood_requests` rows → fetch `units_needed, contact_phone, lat, lng`
  by id, client-side. No migration / RPC.
- Phone reveal is UI-gated behind "I'll help" — consistent with the existing
  feed (requester phone shown only after responding). [[privacy]]

## Tasks

### Task 1 — `src/index.css`
- Add `@keyframes bhPulse` (token ring pulse) and `@keyframes bhModalIn`
  (entrance), helper classes `.bh-pulse` / `.bh-modal-in`, and a
  `@media (prefers-reduced-motion: reduce)` override → `animation: none`
  (accessibility floor).

### Task 2 — `src/components/IncomingRequestAlert.tsx` (new)
- Props: `{ alert, lang, donorLat, donorLng, onHelp(requestId), onClose }`.
- Internal `helped` state (locked → revealed). Fetch `blood_requests` by id
  (units_needed, contact_phone, lat, lng) in a cancellation-guarded `useEffect`.
- Distance via inline haversine from donor coords → `formatDistanceLabel`.
- Render: scrim + centered card; pulsing blood-type token; urgent pill + headline;
  detail card (location / units / distance, hiding rows with no data); phone row
  (locked masked → revealed `formatPhone`); success strip when helped; actions
  (locked: "ကူညီမည်" → onHelp + reveal, "ယခု မကူညီနိုင်ပါ" → onClose;
  revealed: tel: "ဖုန်းခေါ်ရန်", "ပိတ်ရန်" → onClose). EN + MY strings.

### Task 3 — `src/screens/Home.tsx`
- Replace the inline `fcmDonorAlert &&` overlay with
  `<IncomingRequestAlert alert={fcmDonorAlert} lang donorLat donorLng
  onHelp={(id)=>onRespond?.(id)} onClose={()=>onDismissFcmDonorAlert?.()} />`.
- Drop now-unused imports if any (verify Badge still used elsewhere first).

## Done when
- `npm run build` + `npm run lint` clean for changed files.
- Donor notification tap shows the new centered modal; phone stays masked until
  "I'll help", then reveals with a Call button; reduced-motion disables anim.

## Deviations / follow-ups
- "Can't help right now" dismisses only (no `request_responses` decline row) —
  recording declines is inert until re-ping suppression exists. Follow-up.
- Executed directly by orchestrator (design is a DesignSync MCP resource).
